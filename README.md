# Mass Transfer Lab – Cross-check Tool (Mobile + PC)

This is a WhatsApp-link friendly Streamlit web app.
- Faculty select Experiment (1–6)
- Enter ONLY experimental readings
- App prints tables, graphs, and results
- Download Excel + PNG graphs

## Run locally (for you)
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy (to get a shareable link)
1. Create a GitHub repository.
2. Upload `app.py` + `requirements.txt`.
3. Deploy using Streamlit Community Cloud.
